# Bachelor Thesis – AI Workbench

## Abstract

This Bachelor Thesis focuses on the design and implementation of an AI Workbench
that enables users to experiment with AI models through a modular and extensible
web-based platform.

## Project Overview

The system consists of two main components:

- A frontend web application for user interaction
- A backend service providing a REST API and core logic

## Architecture

The frontend communicates with the backend via a REST API.
Both components are developed and deployed independently.

## Technologies

- React (Frontend)
- Node.js & Express (Backend)
- REST API
- Git & GitHub

## How to Run the Project

See the instructions in the `frontend` and `backend` folders.
